README

I. Lancer le programme

Afin de lancer le programme, il suffit de compiler et lancer Program.java

II. Controles du jeu

a) Touche directionnelle droite: bouger vers la droite

b) Touche directionnelle gauche: bouger vers la gauche

c) Touche directionnelle haut: faire tourner le vehicule vers le haut

d) Touche directionnelle bas: faire tourner le vehicule vers le bas

e) Barre espace: faire changer d'orientation le vehicule

f) Touche Z: tirer des missiles si dans un tank

g) Touche E: faire sauter le vehicule

h) Touche Shift: freiner le vehicule

i) Touche R: remettre le niveau actuel a zero

III. Niveaux du jeu

a) Niveau 1 - Bike game originel

Afin de terminer ce niveau, il suffit d'arriver a la ligne d'arrivee

b) Niveau 2 - Wall game

Afin de terminer ce niveau, il faut se frayer un chemin a travers le mur pour arriver a la ligne d'arrivee

c) Niveau 3 - Slay Santa Boss

Afin de terminer ce niveau, il faut detruire le boss du niveau, le pere Noel
